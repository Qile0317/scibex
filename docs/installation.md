# Installation

## Stable release

To install scibex, run this command in your terminal:

```sh
uv add scibex
```

Or if you prefer to use `pip`:

```sh
pip install scibex
```

## From source

The source files for scibex can be downloaded from the [Github repo](https://github.com/Qile0317/scibex).

You can either clone the public repository:

```sh
git clone git://github.com/Qile0317/scibex
```

Or download the [tarball](https://github.com/Qile0317/scibex/tarball/main):

```sh
curl -OJL https://github.com/Qile0317/scibex/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd scibex
uv pip install .
```
